
# Jogo de Reflexos - Detona Ralph

Este projeto é um jogo de reflexos inspirado no universo do Detona Ralph, desenvolvido com HTML, CSS e JavaScript. O jogador deve clicar rapidamente nos quadrados onde o "inimigo" aparece, acumulando pontos a cada acerto. O jogo possui um tempo limite de 30 segundos, após o qual o placar final é exibido. 

## Funcionalidades

- **Aparição aleatória do inimigo**: O inimigo aparece em posições aleatórias da grade a cada intervalo, aumentando o desafio.
- **Placar em tempo real**: A pontuação do jogador é atualizada a cada acerto.
- **Contador de tempo**: Mostra o tempo restante para o fim do jogo.
- **Efeito sonoro**: Um som de acerto é reproduzido a cada clique correto.
  
## Como Jogar

1. Abra o jogo em seu navegador.
2. Clique nos quadrados onde o inimigo aparecer para marcar pontos.
3. Jogue o mais rápido possível antes que o tempo acabe!

## Tecnologias Utilizadas

- HTML, CSS e JavaScript
